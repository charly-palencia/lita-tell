Lita plugin to send a message to a specific user or  room

## Installation

Add lita-tell to your Lita instance's Gemfile:

``` ruby
gem "lita-tell"
```

## Usage

```
lita tell [user|room] hi sir!
lita say "hi sir!" to [user|room]
```
